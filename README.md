# About Focusbili PROJ
Focusbili是一个为了极端专注场景打造的视频流载体。它将特定的网页端（space.bilibili.com/uid）打包为独立的app容器，通过代码拦截乱逛，把制作的app——并入Android锁机程序(比如番茄todo)——实现极限自律

本项目代码支持gradle批量生成不同uid uploader的app

由于技术问题，目前仅支持把单个app浏览空间局限在单个uploader的空间space内

严禁用于任何非法侵入或非法控制计算机系统的行为
